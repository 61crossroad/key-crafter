## key-crafter

'키크래프터'는 기계식 키보드와 주변기기를 판매하는 온라인 쇼핑몰입니다.
아래 링크에서 테스트 가능합니다.

http://13.209.232.233:8080/

테스트 아이디/비밀번호 : aaa/aaaa
또는 계정 생성도 가능합니다.
